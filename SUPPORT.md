# Support

Use GitHub Issues for bug reports and feature requests.

Before opening an issue:

```bash
ruff check src tests
pytest
wq-agent status
```

When asking for help, include:

- wq-agent version or commit.
- Python version and operating system.
- The command you ran.
- Expected behavior.
- Actual behavior.
- Sanitized logs or traceback.

Do not include credentials, account identifiers, private alpha records, local
databases, or private wiki pages.
