# `winsorize`

**Category**：Cross Sectional　**Type**：SCALAR

## 签名

```
winsorize(x, std=4)
```

## 官方说明

Winsorize limits values in a data to within a specified number of standard deviations from the mean, reducing the impact of extreme outliers.
