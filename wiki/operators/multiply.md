# `multiply`

**Category**：Arithmetic　**Type**：SCALAR

## 签名

```
multiply(x ,y, ... , filter=false), x * y
```

## 官方说明

Multiplies two or more inputs element wise. Set filter=true to treat NaNs as 0 before multiplication
