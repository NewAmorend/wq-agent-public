# `sqrt`

**Category**：Arithmetic　**Type**：SCALAR

## 签名

```
sqrt(x)
```

## 官方说明

Returns the non negative square root of x. Equivalent to power(x, 0.5); for signed roots use signed_power(x, 0.5).
