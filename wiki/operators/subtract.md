# `subtract`

**Category**：Arithmetic　**Type**：SCALAR

## 签名

```
subtract(x, y, filter=false), x - y
```

## 官方说明

Subtracts inputs left to right: x ? y ? … Supports two or more inputs. Set filter=true to treat NaNs as 0 before subtraction.
