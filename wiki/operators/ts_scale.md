# `ts_scale`

**Category**：Time Series　**Type**：SCALAR

## 签名

```
ts_scale(x, d, constant = 0)
```

## 官方说明

Scales a time series to a 0–1 range based on its minimum and maximum values over a specified period, with an optional constant shift.
